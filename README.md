## Features

* Add and manage sales records
* Dashboard view for analytics
* Upload and store data
* Generate reports

##  Tech Stack

* HTML, CSS
* Python (Flask)
* SQLite Database

##  Project Structure

* `templates/` → Frontend pages
* `app.py` → Backend logic
* `sales.db` → Database

##  Purpose

## 🎯 Purpose

The purpose of this project is to develop a smart Retail Sales Management System that not only stores and manages sales data but also provides meaningful business insights and decision support.

This system helps businesses track sales performance, identify profit and loss trends, analyze category-wise performance, and make data-driven decisions. It includes features like dashboard analytics, AI-based insights, and a what-if simulator to predict the impact of pricing and cost changes.

The goal is to transform raw sales data into actionable insights, enabling better business strategy and performance optimization.
