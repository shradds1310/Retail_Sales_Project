## Retail Sales Management System

A web-based application to manage retail sales, track records, and visualize business performance.

## Features

* Add and manage sales records
* Dashboard view for analytics
* Upload and store data
* Generate reports

##  Tech Stack

* HTML, CSS
* Python (Flask)
* SQLite Database

##  Project Structure

* `templates/` → Frontend pages
* `app.py` → Backend logic
* `sales.db` → Database

##  Purpose

This project helps small businesses manage and analyze their sales efficiently.
